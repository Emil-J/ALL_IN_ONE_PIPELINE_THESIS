DID THE SAME FOR ROADS.


Currently I am increasing the width of each road and will do the same for each waterway.
As I am manually analyzing each road category in OSM I am thinking about a way of having more classes for each segment. OFFROAD, STREET ROADS, MOTORWAY, FOOTWAY. For the model to segment what he sees in a better fashion and then also to create a possibility for this to be used to deduce the region of the locaton a bit more since there is more differentiation that the model can work with. This is just an idea.



For this to be possible I will write down all what I analyze based on each category in OSM as it seems to be very valuable to write down:


roads_unclassified.osm_lines 	Unclassified category of roads, they include cities and country roads along with sometimes tracks or paths for agriculture or hiking. Can't really use this class for anything since it isn't organized. Label ROAD
roads_track.osm_lines	Class for agriculture motorized or forest hiking/motorized tracks, i think it should be labeled for OFFROAD, since it is always seen this class in country/forest regions and most of the time they are gravel/dirt roads which are highly distinguishable visually.
roads_residential.osm_lines	Class for street roads in residential areas. Label ROAD
roads_motorway.osm_lines	Class for Motor Way roads. Label ROAD
roads_footway.osm_lines	Class for foot paths for pedestrians in cities/towns/villages. Very inconsistent category, graveyard paths and residential paths around apartments are the most classified ones. Label ROAD
roads_cycleway.osm_lines	Class for cyclist roads. Unusable, since it is always next to roads and wide as a car, model would be too confused and would misclassify. Label ROAD
roads_path.osm_lines	Class for hiking paths, walking paths, motorized offroad paths, offroad cycling paths. Label OFFROAD
roads_primary.osm_lines	Class for primary roads? Label ROAD
roads_pedestrian.osm_lines	Class for pedestrian roads in cities, although Denmark sometimes still has cars there. So, Label ROAD
roads_living_street.osm_lines	Class for roads, living street, apartment areas, not sure. Label ROAD
roads_secondary.osm_lines	Class for secondary motorway roads? Label ROAD
roads_service.osm_lines	Class for service roads in cities or roads from main roads to residential houses. Label ROAD
roads_tertiary.osm_lines	Class for teritiary motorway roads? Label ROAD
roads_raceway.osm_lines	Class for raceway roads. Label ROAD.

When Focal CE helps
Focal Cross-Entropy down-weights “easy” pixels and focuses training on hard / misclassified pixels. That’s useful when:

When Focal CE doesn’t help much

•	noisy labels / inconsistent masks
•	not enough resolution/context (256×256 too small)
